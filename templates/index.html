How our model works

01
Market Research & Data Generation

2026 property rates from 99acres, NoBroker & Coldwell Banker + construction costs from industry sources

02
12-Feature Engineering + Scaling

StandardScaler normalizes all features — Area, BHK, Bath, Location, Property Type + Construction, Architect, Engineer, Approval, Utility, GST, Resale

03
Hyperparameter-Tuned GBR

RandomizedSearchCV (30 iterations, 5-fold CV) finds optimal n_estimators, max_depth, learning_rate, subsample, and more

04
Price Range + AI Description API

Flask /predict returns price with ±5-8% confidence bounds and auto-generated property description
